Answer
Gradient descent is an iterative first-order optimisation algorithm that minimises a differentiable loss function by updating model parameters in the direction of the steepest negative gradient. At each iteration, the parameters θ are adjusted as θ ← θ − η∇L(θ), where η is the learning rate and ∇L(θ) is the gradient of the loss with respect to θ. The algorithm converges progressively toward the local minimum of the cost surface.
